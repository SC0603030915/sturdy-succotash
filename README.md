# Global GDP Trends Analysis (1990-2024)

## Project Description
This project uses World Bank Open Data to analyze GDP growth trends of major global economies from 1990 to 2024, comparing the development patterns between developed and emerging economies.

## Research Questions
1. How have the GDP totals of major economies changed over the past 30 years?
2. What are the differences in growth patterns between developed and emerging economies?
3. Is the gap in GDP per capita widening or narrowing?

## Target Audience
- Economics students: Understanding changes in the global economic landscape
- Policy researchers: Obtaining economic data comparisons across countries
- General public: Visually understanding world economic trends

## Dataset
- **Source**: World Bank Open Data
- **Access Date**: April 2026
- **Time Range**: 1990-2024
- **Countries**: United States, Japan, Germany, China, India, Brazil
- **Indicators**: GDP (current US$), GDP growth (annual %), GDP per capita (current US$)

## How to Use
1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run Jupyter Notebook: `jupyter notebook notebook/gdp_analysis.ipynb`
4. Execute all cells in order

## Key Findings
- China's economic growth has been the most significant
- Emerging economies generally show higher growth rates than developed economies
- The 2008 financial crisis and the 2020 pandemic caused noticeable shocks to the global economy

## Tech Stack
- Python 3.14
- pandas: Data processing
- matplotlib & seaborn: Data visualization
- wbgapi: World Bank data acquisition
